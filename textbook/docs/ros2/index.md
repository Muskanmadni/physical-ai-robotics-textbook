# Module 1: ROS 2

This module covers the Robot Operating System version 2 (ROS 2), which is a flexible framework for writing robot software. ROS 2 is a collection of tools, libraries, and conventions that aim to simplify the task of creating complex and robust robot behavior across a wide variety of robot platforms.

## Topics Covered

- Introduction to ROS 2
- Nodes and Topics
- Services and Actions
- Practical Examples

This module contains several chapters that will guide you through ROS 2 concepts and practical applications.

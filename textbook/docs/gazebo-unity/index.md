# Module 2: Gazebo / Unity Simulation

This module explores simulation environments for robotics, focusing on Gazebo and Unity. Simulation is a critical aspect of robotics development, allowing for testing and validation before deploying to real robots.

## Topics Covered

- Gazebo Basics
- Unity Simulation
- Comparison Study

This module provides hands-on experience with leading simulation platforms in robotics.
